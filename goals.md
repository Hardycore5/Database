## My Learning Goals

1. Understand what a database is and why it is important in the real world — 
   including how apps like Instagram and WhatsApp use databases to store information.

2. Learn how to write basic SQL commands: SELECT (to read data), INSERT (to add data), 
   UPDATE (to change data), and DELETE (to remove data).

3. Understand the difference between SQL and NoSQL databases — 
   when to use each one and what makes them different from each other.
