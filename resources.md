## Helpful Resources

1. [W3Schools SQL Tutorial](https://www.w3schools.com/sql/) — Great for learning SQL step by step.
2. [freeCodeCamp Database Course](https://www.freecodecamp.org) — Free full course on databases.
3. [Khan Academy Computing](https://www.khanacademy.org/computing) — Easy beginner explanations.
4. [W3Schools SQL Joins](https://www.w3schools.com/sql/sql_join.asp) 
   — Explains how tables are connected to each other using keys.
5. [SQLZoo](https://sqlzoo.net) 
   — A free website where you can practise writing real 
   SQL commands directly in your browser. Great for beginners!
6. [W3Schools SQL Index](https://www.w3schools.com/sql/sql_create_index.asp)
   — Explains what database indexes are and how they 
   help speed up data retrieval in large databases.
